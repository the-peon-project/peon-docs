## Interface Features

The PEON admin interface provides:
- **Server Management** - Create, start, stop, and configure game servers
- **Real-time Monitoring** - View server status, resource usage, and player counts  
- **Configuration Tools** - Modify server settings and game-specific options
- **Backup Management** - Schedule and restore server backups
- **Log Viewing** - Monitor server logs and troubleshoot issues

*Coming in PEON v3.1.0*
